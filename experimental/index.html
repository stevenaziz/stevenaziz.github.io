Academic & Personal Projects

Expression Simplifer

Individual Data Structures I

Using C++, implemented a postfix expression simplifier that reads in an algebraic expression written in postfix and turns it into a binary expression tree. The simplifier recognizes both numbers and variables, and can perform basic operations such as + (addition), - (subtraction), and *(multiplication).

Simple List Web App

Group Netcentric Computing

Using HTML, CSS, JavaScript, and Node.js, implemented both front-end and back-end components of a simple list web app that has the ability to add data to a simulated database via a JSON file, remove data from the "database", and load the data each time the page loads. The Node.js module Express was used to support the backend.
